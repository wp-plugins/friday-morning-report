Dislays an Archive of Friday Morning Report shows.

== Description ==

This plugin displays an archive of shows from the <a href="http://fridaymorningreport.tv">Friday Morning Report</a> show broadcast every Friday at 9:15 CST.
The host of the show is <a href="http://craig.cmehil.com">Craig Cmehil</a>.

== Author Homepage ==

http://blog.abesh.net 

== Installation ==

1. Upload the `fmr-wp-plugin` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
