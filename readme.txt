=== Chinposin ===
Contributors: abesh
Tags: chinposin, avatar, slideshow
Requires at least: 2.6
Tested up to: 2.6.3
Stable tag: trunk

Displays a random slideshow of all the avatars that a user has registered on chinposin.com

== Description ==

What started off as an one off experiment is growing into a full fledged avatar management platform with more and more functionality being added every day. With a avatar timeline per user along with it’s feed, Tiltviewer and many more upcoming features all of which can be found on the <a href="http://chinposin.wordpress.com/">Chinposin Blog</a>.

== Installation ==

1. Upload the `wp_chinposin` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
